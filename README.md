This project automates the insurance claims assessment interview process using Large Language Models (LLMs) and LangChain. The system conducts a conversational interview with claimants, extracts structured claim information, validates responses, identifies potential inconsistencies, performs fraud risk assessment, and recommends the next course of action.

The goal is to reduce manual effort, improve data quality, and accelerate claim processing while ensuring that high-risk claims are flagged for human review.
